# Programming Project 2 from CS4310 for Prof. Gilbert Young

# CS 4310 - Operating Systems


Simulates three page replacement algorithms in Java:
- **FIFO** (First In First Out)
- **LRU** (Least Recently Used)
- **OPT** (Optimal Algorithm)

Each algorithm reads 50 reference strings of length 30 from `TestingData.txt`
and runs them against four frame sizes (3, 4, 5, 6), printing a detailed
frame table for each string and reporting the average page faults per frame size.

**Language:** Java
